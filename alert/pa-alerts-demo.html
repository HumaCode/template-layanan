PA Alerts — SIPEKA Notification System

Library notifikasi & dialog khusus SIPEKA. Toast, dialog, confirm, prompt, loading, dan inline alert dengan animasi smooth dan ikon interaktif.

import pa-alerts.js  →  PA.toast()  ·  PA.dialog()  ·  PA.confirm()
1

Toast Notifications

PA.toast()
Success Toast
Notifikasi berhasil — data tersimpan, aksi berhasil dieksekusi.
PA.success('Data berhasil disimpan');
Error Toast
Notifikasi gagal — koneksi error, validasi gagal, dsb.
PA.error('Koneksi ke server gagal');
Warning Toast
Peringatan — sesi hampir habis, data tidak lengkap, dsb.
PA.warning('Sesi Anda akan berakhir');
Info Toast
Notifikasi informasi — pembaruan sistem, info umum.
PA.info('Pembaruan sistem tersedia');
Teal Toast
Custom teal — notifikasi aksi khusus atau highlight.
PA.toast({type:'teal',message:...});
Custom Toast
Custom icon, durasi panjang, posisi atas-kanan.
PA.toast({
type:'purple',duration:8000,
position:'top-right'
});
Posisi Toast
Klik posisi untuk melihat toast muncul di lokasi tersebut.

Dialog Modal

PA.dialog()
Info Dialog
Dialog informasi umum dengan satu tombol tutup.
PA.dialog({
type:'info',
title:'Informasi Sistem',
message:'Pesan info...'
});
Success Dialog
Dialog berhasil — konfirmasi setelah aksi selesai.
PA.dialog({
type:'success', icon:'save',
buttons:[{label:'Lanjut',...}]
});
Warning Dialog
Dialog peringatan — aksi berisiko, perhatian diperlukan.
PA.dialog({
type:'warning', icon:'lock',
title:'Akses Terbatas'
});
Multi-Action Dialog
Dialog dengan beberapa pilihan tombol — Promise resolve ke label tombol.
PA.dialog({
buttons:[
{label:'Simpan Draft',...},
{label:'Publish',...}
]
});
Klik tombol untuk melihat hasil Promise...

Confirm Dialog

PA.confirm()
Hapus Data
Konfirmasi sebelum menghapus data — resolve(true/false).
const ok = await PA.confirm({
type:'error', icon:'delete',
confirmLabel:'Ya, Hapus',
confirmClass:'pa-btn-danger'
});
Menunggu konfirmasi...
Konfirmasi Logout
Konfirmasi sebelum keluar dari sistem.
PA.confirm({
type:'warning', icon:'logout',
title:'Keluar dari Sistem?'
});
Menunggu konfirmasi...
Konfirmasi + Checkbox
Tombol Konfirmasi hanya aktif setelah checkbox dicentang — untuk aksi berbahaya.
PA.confirm({
requireCheck: true,
checkLabel:'Saya memahami...'
});
Menunggu konfirmasi...
Reset Password
Konfirmasi reset password pengguna dengan ikon key.
PA.confirm({
type:'info', icon:'key',
confirmClass:'pa-btn-primary'
});
Menunggu konfirmasi...

Prompt Input Dialog

PA.prompt()
Input Sederhana
Input teks sederhana — resolve ke string yang dimasukkan user.
const val = await PA.prompt({
title:'Nama Kategori Baru',
label:'Nama',
placeholder:'Masukkan nama...'
});
Menunggu input...
Input + Validasi
Input dengan custom validate function — reject jika format salah.
PA.prompt({
validate(v){
if(!v.includes('@'))
return 'Email tidak valid'
}
});
Menunggu input...
Rename / Edit Nama
Prompt dengan nilai default — untuk fitur rename atau edit cepat.
PA.prompt({
title:'Rename Layanan',
defaultValue:'Surat Keterangan',
confirmLabel:'Simpan Nama'
});
Menunggu input...
Verifikasi NIP
Input NIP dengan validasi 18 digit angka.
PA.prompt({
label:'NIP (18 digit)',
inputType:'text',
validate: v => ...
});
Menunggu input...

Loading Overlay

PA.loading()
Loading Dasar
Loading overlay dengan auto-close setelah 2 detik.
const ld = PA.loading({
title:'Memuat...'
});
// nanti:
ld.close();
Loading Upload
Loading dengan pesan dinamis — update via setMessage().
ld.setTitle('Mengunggah...');
ld.setMessage('File 2/3...');
Loading + Batalkan
Loading yang bisa dibatalkan — closable: true.
PA.loading({
title:'Memproses...',
closable:true
});

PA.alert()
Inject ke Elemen
Alert yang diinjeksikan ke dalam elemen DOM — cocok untuk form validation atau feedback di dalam page.

Skenario Nyata

Alur lengkap PA Alerts
Simpan Formulir
Alur: Loading → API call simulasi → Success dialog → Toast.
// loading → sukses
const ld = PA.loading(...);
// await API...
ld.close(() => PA.dialog(...));
Hapus User
Alur: Confirm → Loading → Success toast atau Error.
if(await PA.confirm(...)){
const ld = PA.loading(...);
// await API...
ld.close(() => PA.success(...));
}
Approve Permohonan
Alur: Prompt catatan → Confirm → Loading → Success.
const catatan = await PA.prompt(...);
if(catatan && await PA.confirm(...)){
// proses...
}
Ganti Password
Alur: Prompt password baru → Validasi → Confirm → Loading → Success.
const pwd = await PA.prompt({
inputType:'password',
validate: v => v.length < 8
? 'Min 8 karakter' : null
});
